# Dashboard Documentation
## Project 5 — Smart Document Automation: DG & Customs Clearance

**Tool:** Tableau Cloud (dub01.online.tableau.com)  
**Workbook:** ShipmentDoc-Compliance-Automation  
**Dataset:** data/raw/shipments.csv (500 records)  
**Note:** Tableau Cloud was used instead of PowerBI Desktop — PowerBI Desktop is Windows-only and not available on macOS. PowerBI web requires a work/school Microsoft account. Tableau Cloud provides equivalent functionality.

---

## Use Case Description
A medium-sized German freight forwarder (50–200 employees) processes 50–300 international shipments daily. Each shipment generates multiple compliance-critical documents — both Dangerous Goods (DG) declarations and Customs Clearance paperwork. This dashboard gives management a live view of AI compliance automation performance across both document types.

---

## Dashboard 1 — ShipmentDoc Compliance Automation (Overview)

### Key Metrics Explained

| Chart | What it shows | Why it matters |
|---|---|---|
| [DG] Doc Type Breakdown | Count of each document type processed | Shows which documents the AI handles most — Customs Export Declarations (184) are highest volume |
| [ALL] DG vs Customs | Split between Dangerous Goods (165) vs Customs (212) docs | Proves the system covers BOTH compliance areas, not just one |
| [ALL] Compliance Status | Auto-approved (276) vs Human review (218) vs Flagged (6) | Shows AI accuracy — only 1.2% flagged vs 5–10% industry average |
| [ALL] Processing Time | Average AI processing time per document type (seconds) | Shows AI speed — avg 44s vs 603s manual = 93% time reduction |
| [ALL] Confidence Score | Average AI confidence score per document type (%) | Shows AI reliability — avg 90.7%, Commercial Invoice highest (93%) |
| [ALL] Time Saved | Average seconds saved per document type vs manual baseline | Directly shows ROI — avg 559s saved per document |

### Design Rationale
- **Communication layer focus**: Charts show KPIs a CEO can understand immediately (time saved, error rate, confidence) rather than raw technical data
- **Horizontal bar charts**: Used throughout for easy comparison across document types
- **Color coding by doc type**: Consistent colors across all charts so Chleo can track one document type across multiple views
- **Axis ranges**: Confidence score axis starts at 80% (not 0%) to show meaningful variation in the 87–93% range

### How to Use
1. Go to https://dub01.online.tableau.com
2. Open workbook: ShipmentDoc-Compliance-Automation
3. Click any bar to filter all charts by that document type
4. Use the legend on the right to highlight specific document types

## Dashboard 2 — Customs Clearance Dashboard

### Key Metrics Explained

| Chart | What it shows | Why it matters |
|---|---|---|
| [CUSTOMS] Doc Types | Count of each customs document type | Shows CMR Waybills (89) and Export Declarations (184) dominate customs work |
| [CUSTOMS] Compliance Status | Auto-approved vs Human review for customs docs only | Shows customs-specific compliance rate — no flagged customs docs |
| [CUSTOMS] Routes | Which international routes generate most customs documents | DE→TR (43), DE→US (41), DE→GB (37) are highest priority for automation |
| [CUSTOMS] Processing Time | Avg AI processing time per customs document type | Export Declarations take longest (~44s) due to HS code and EORI validation |

### Design Rationale
- **Separate customs dashboard**: Isolates customs compliance from DG to give Chleo focused insights per workstream
- **Route chart**: Directly actionable — shows which routes need the most automation attention
- **Sorted bars**: Routes sorted by volume so highest priority appears first

## Data Source Information
- **File**: data/raw/shipments.csv
- **Records**: 500 synthetic shipment documents
- **Generated by**: generate_dataset.py
- **Columns**: doc_id, date, doc_type, route, carrier, processing_time_seconds, manual_baseline_seconds, time_saved_seconds, confidence_score, compliance_status, error_detail, is_dangerous_goods, dg_class, un_number, confidence_band, doc_category
- **Routes covered**: DE→CH, DE→CN, DE→FR, DE→GB, DE→IT, DE→JP, DE→NL, DE→PL, DE→TR, DE→US
- **Document types**: Customs Export Declaration, CMR Waybill, DG Declaration (ADR), DG Declaration (IMDG), DG Declaration (IATA), Commercial Invoice, EUR.1 Certificate
